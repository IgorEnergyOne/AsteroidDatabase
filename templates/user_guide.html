Parameters and Their Explanations

Parameter Explanation
data: 'id_num' This parameter defines the column that will display the 'id_num' field from each row of data.
data: 'number' This column will show the value of the 'number' field from each row of data.
data: 'name' This parameter maps the 'name' field to the respective column.
data: 'class', orderable: false Displays the 'class' field, and disables sorting on this column (the column cannot be clicked to sort).
data: 'a', name: 'a' Displays the 'a' field and assigns it a custom name 'a' for internal use, such as for referencing or filtering.
data: 'e', name: 'e' Displays the 'e' field and gives it a custom internal name 'e'.
data: 'i', name: 'i' Displays the 'i' field and assigns it a custom internal name 'i'.
data: 'orbital_period' This parameter binds the 'orbital_period' field to the respective column.
data: 'family_number' This parameter binds the 'family_number' field to the respective column.
data: 'family_name', orderable: false Displays the 'family_name' field and disables sorting for this column.
data: 'albedo' This parameter binds the 'albedo' field to the respective column.
data: 'mag' This parameter binds the 'mag' field to the respective column.
data: 'density' This parameter binds the 'density' field to the respective column.
data: 'diameter' This parameter binds the 'diameter' field to the respective column.
data: 'taxonomy class', orderable: false Displays the 'taxonomy class' field and disables sorting for this column.
data: 'thermal inertia' This parameter binds the 'thermal inertia' field to the respective column.
data: 'spin' This parameter binds the 'spin' field to the respective column.
